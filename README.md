# BookingWebScraping_EventTimePrices

This project prepares a reproducible Python environment for web scraping and basic data analysis using uv.

## Setup

Install runtime dependencies:

    uv sync

Install development tools (for Jupyter / VS Code notebooks):

    uv sync --dev

## Python Version

This project uses the following Python version range:

    >=3.12,<3.13

## Included Libraries

Runtime dependencies:
- requests
- beautifulsoup4
- lxml
- selenium
- webdriver-manager
- numpy
- pandas
- pyarrow

Development dependencies:
- ipykernel
